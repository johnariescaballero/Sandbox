# 📊 Data Analysis Sandbox

Welcome! This repository serves as my personal laboratory for mastering the full data lifecycle—from raw extraction and cleaning to advanced modeling and interactive visualization. 

## 📂 Repository Structure

* **[SQL Sandbox](./SQL%20Sandbox):** Scripts for database schema design (**DDL**) and data manipulation (**DML**). Focuses on complex joins, CTEs, and Exploratory Data Analysis (EDA).
* **[Python Sandbox](./Python%20Sandbox):** Automated data cleaning and statistical analysis using **Pandas** and **NumPy**, along with data visualization scripts. 
* **[Power BI Sandbox](./PowerBI%20Sandbox):** End-to-end business intelligence reports featuring interactive dashboards and advanced **DAX** measures.
* **[Excel Sandbox](./Excel%20Sandbox):** Advanced data modeling using the **Power Suite**:
    * **Power Query (M):** ETL processes for cleaning and transforming disparate data sources.
    * **Power Pivot:** Building relational data models and star schemas within Excel.
    * **DAX:** Creating sophisticated calculated columns and measures for KPI tracking.

---

## 🛠️ Tech Stack & Skills

### **Data Modeling & Calculations**
* **DAX (Data Analysis Expressions):** Time intelligence, filter context manipulation (`CALCULATE`), and iterator functions (`SUMX`).
* **Power Query:** Data merging, pivoting/unpivoting, and custom transformations.
* **Power Pivot:** Managing large-scale data models and table relationships.

### **Programming & Databases**
* **SQL (DDL/DML):** MySQL (Joins, CTEs, Window Functions, Table Creation).
* **Python:** Data cleaning, transformation, and statistical analysis.

### **Visualization & Reporting**
* **Power BI:** Dashboard design and storytelling.
* **Excel:** Dynamic Pivot Tables and advanced charting.

This sandbox is updated regularly as I explore new datasets and refine my analytical workflow. My current focus is on optimizing SQL queries for performance and building more complex relational models in Excel and Power BI.
